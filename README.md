# Benchmark International School – A School Website

Benchmark International School is a modern educational website created using HTML, CSS, and Bootstrap. The site showcases the school’s mission, leadership team, and overall educational vision. It is a static website hosted using GitHub Pages and provides visitors with an informative online experience about the school.

###  Live Website: 
https://firecoder29.github.io/Benchmark-Intl.-School/

---

## Features
- Clean and responsive layout
- Navigation menu for easy browsing
- Displays information about school leadership
- Highlights the school’s motto and educational focus
- Includes images and sections for principals and administrators
- Fast loading and mobile-friendly design

---

## Content Sections
- Home page with school overview
- About section for school leaders
  - Principal  
  - Vice Principal  
  - Chairman  
  - Chief Executive Officer
- Image gallery showing school visuals
